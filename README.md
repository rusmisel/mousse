# mousse
vim for your mouse.

## installation
### Arch Linux
avaliable [on the AUR](https://aur.archlinux.org/packages/mousse-git)

clone and build or run `paru -S mousse-git`

### Source
`make install` (installs to `~/.local/bin` by default)

depends on `wayland-protocols`, `wlr-protocols`, `clang`, `wayland-scanner`
